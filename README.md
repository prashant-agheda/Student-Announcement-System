## About The Project

This is and Android App with Firebase Authentication which helps to do Google Sign In as well as Facebook Login. It also allows to Update the User Profile using
Storage in Firebase and also the data of Announcements is saved in Realtime Database.

Features
* Authentication
* Storage
* Realtime Database

### Built With

* [Android Developer Guide](https://developer.android.com/guide)
* [Material Design](https://material.io/)
